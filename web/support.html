Kemetic Calendar Support

For help with Kemetic Calendar, email support@maat.app.

Please include the device model, app version, platform, and a short description of what happened. If you are reporting account, login, reset-link, calendar sync, notification, sharing, or profile issues, include the email address on the account when it is safe to do so.

For privacy, data access, correction, deletion, moderation, report, or block-related requests, use the same support address.

Basic troubleshooting: update the app, confirm you are signed in, check notification or calendar permissions in device settings, and try again on the same device if you are using a password reset link.

Account deletion: maat.app/delete-account

Privacy policy: maat.app/privacy

Terms and conditions: maat.app/terms
